# Matplotlib Essentials

This repository contains a Jupyter Notebook (`matplotlib_essentials.ipynb`) focused on the fundamentals of data visualization in Python using Matplotlib.

## Overview
* **Introduction to Matplotlib:** Background, object-oriented API, and language breakdown (Python, C++, etc.).
* **Plotting Basics:** Drawing markers and connecting points on X and Y axes using `pyplot`.
* **Format Strings (`fmt`):** Utilizing shortcut string notation to define markers, line styles, and colors.
* **Marker Customization:** Adjusting marker size (`ms`), edge color (`mec`), and face color (`mfc`).
* **Random Data Generation:** Using NumPy to generate and plot standard normal distribution data.
